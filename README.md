# SVP-to-LRC-tool
一个可以将你的svp工程转换成lrc歌词的本地Web应用

*本项目代码借助AI编写*

要求你有已经分好行的歌词（与工程内的词的字数要完全对的上，工程内的词可以用别的音近词或拼音代替）

目前仅限于纯中文填词的svp文件，建议将歌词包含的音符单独整合到一个新svp文件的单个轨道，svp文件除“br”、“-”、“+”外尽量不要包含其他符号，也不要在一个音符块内填入两个及以上的词。别问条件为什么这么苛刻，问就是因为本人菜，太复杂的我做不出来（）

v0.0.0版本下载后双击index.html即可在浏览器中打开使用
v1.0.0版本由于使用了ES6模块，在本地运行时需要通过HTTP服务器来加载才能获得完整体验，不能直接通过file://协议打开。下载后双击start.bat（windows）start.sh（linux）在弹出的网页中即可使用，前提要求安装Python并添加到PATH
